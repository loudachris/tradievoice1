# Loudachris Brand Guidelines

## Description
This global skill defines the strict visual and tonal identity for the Loudachris high-end Agency Lead Magnet. It serves as the source of truth for all design decisions, ensuring a cohesive, premium, and professional user experience across all interfaces.

## Triggers
- Agency Lead Magnet
- Loudachris brand
- Designing UI components
- Writing copy for the agency
- Creating marketing assets

## Review Checklist
- [ ] Primary buttons are styled with Brilliant Rose (#E357AB)
- [ ] Secondary elements (Toasts/Icons) use Tropical Indigo (#8C87C9)
- [ ] Background is Platinum White (#F5F5F5)
- [ ] Text content is Dark Navy (#122933)
- [ ] Typography is exclusively 'Filson Soft' or 'Nunito'
- [ ] Logo is present and fixed in the Top-Left corner
- [ ] Tone is consistently Professional, Direct, and High-Value

## Loudachris Design Guidelines
**STRICT ADHERENCE REQUIRED**

All output and artifacts generated by this skill must comply with the Loudachris Design Guidelines, specifically tailored for this brand:

### 1. Brand Identity & Colors
- **Primary Action (Buttons)**: `#E357AB` (Brilliant Rose)
    - Use for call-to-action buttons and primary interactive elements.
- **Secondary (Toast/Icons)**: `#8C87C9` (Tropical Indigo)
    - Use for accents, icons, and notification states.
- **Background**: `#F5F5F5` (Platinum White)
    - Clean, modern backdrop for all pages.
- **Text**: `#122933` (Dark Navy)
    - High contrast, readable, professional.

### 2. Typography
- **Families**: 'Filson Soft' or 'Nunito'
- **Usage**: Apply consistently for headings and body text.
- **Style**: Clean, legible, and modern.

### 3. Layout & Structure
- **Logo Position**: Always fixed **Top-Left**.
- **Whitespace**: Use generous whitespace to create a feeling of luxury and openness.

### 4. Core Design Principles (Global)
- **Rich Aesthetics**: Use vibrant colors and harmonious palettes (as defined above).
- **Visual Excellence**: Prioritize premium, state-of-the-art design over generic defaults.
- **Dynamic Design**: Interfaces must feel alive with hover effects, micro-animations, and smooth transitions.
- **Premium Feel**: No "Minimum Viable Product" vibes. The result must WOW the user immediately.
- **No Placeholders**: Use real text and generate real assets/images.

### 5. Tone of Voice
- **Attributes**: Professional, Direct, High-Value.
- **Execution**: Communicate authority and expertise without unnecessary fluff. Focus on user benefit and premium service.
